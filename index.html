About This Project

This project demonstrates a complete CI/CD pipeline for a containerized web application. Every code change pushed to GitHub automatically triggers Jenkins to build Docker images, test the application, and deploy it to an AWS EC2 instance using Docker Compose.

CI/CD Workflow

  1. Developer pushes code to GitHub.
  2. GitHub webhook triggers Jenkins.
  3. Jenkins clones the repository.
  4. Docker images are built.
  5. Application is tested.
  6. Terraform provisions infrastructure (if required).
  7. Docker Compose deploys the application.
  8. The updated website is available to users.
